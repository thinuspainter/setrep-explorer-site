Adding a configuration / manager

Each folder is one self-contained explorer. To add one: (1) copy default/ to a new folder named for the config (lowercase, no spaces); (2) replace its index.html with a locked build that has that config preloaded (ask the tool owner to generate it), or leave the upload-enabled build so the manager loads their own export; (3) add a card above pointing at the new folder. Commit and push — GitHub Pages serves it automatically within a minute.
